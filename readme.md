# coaProject

> Build with
>- PHP 7.1
>- Laravel 5.8
>- Maria DB

- [使用農委會開放資料平台](https://data.coa.gov.tw/Default.aspx)提供的資料「[推薦農村優良伴手禮](https://data.coa.gov.tw/Query/ServiceDetail.aspx?id=197)」
整理成比較好閱讀的方式
- 使用者可對各項產品進行評論
- 使用者可發佈並編輯自己的產品
- 使用者可利用API取得所有產品資料
